문자 메세지 파싱 가계부 웹앱

WEB APPLICATION

기획 및 스토리

지출관리에 관심이 많았던 선배가 같이 작은 프로젝트를 제안하였습니다. 선배가 문자메세지를 파싱하는 안드로이드 미들웨어를 개발하였고, 저는 미들웨어에서 넘어온 JSON 형식의 데이터를 처리하는 웹 애플리케이션을 제작하였습니다.

Programing with : MeteorJs(Blaze) / Bootstrap

Period : 2016(1 month)

Code : github

내용 소개

MeteorJS 프레임워크를 Blaze로 작성하고, CSS프레임워크인 BootStrap을 사용하여 화면을 꾸몄습니다. 기본적인 로그인 기능이 있으며 로그인을 할 경우 자신이 지출한 금액을 정렬할 수 있으며, 작성과 삭제를 할 수 있습니다. 날짜 우선으로 기본정렬되며, 월별 지출 목록을 확인할 수도 있습니다. 월별로 얼마나 지출이 많고 적은지 시각적으로도 확인할 수 있습니다.

배운 점 & 부족했던 점

점차 웹 개발에 익숙 해져가는 느낌을 받았습니다. 많이 부족하지만 성취감이 있었기 때문에 더 잘하고 싶어졌고, 더 열심히 공부하였습니다.
안드로이드를 미들웨어만으로 쓰는 것이 아닌, 모바일로도 사용 가능하면 더 편리할 것이기 때문에 웹을 모바일로 변환하는 기술이 있었다면 어땠을까 하는 아쉬움이 남았습니다.
